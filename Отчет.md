# Отчет по курсу системного программирования Закирянов Д.А. 2231117

## Введение

Курс системного программирования предоставил комплексное понимание принципов разработки системного программного обеспечения, включая архитектуру приложений, безопасность, управление данными и взаимодействие с различными уровнями программной структуры. В рамках курса был реализован проект системы управления автобусами и уведомлениями о событиях, демонстрирующий применение изученных концепций.

## Изученные темы

### 1. Архитектура приложений
- **Spring Boot**: Изучены основы создания веб-приложений с использованием фреймворка Spring Boot, включая автоматическую конфигурацию, внедрение зависимостей и управление жизненным циклом бинов.
- **DI/IoC (Dependency Injection/Inversion of Control)**: Понимание концепции внедрения зависимостей и инверсии управления, жизненного цикла бинов и влияния скоупов на объекты в многопоточной среде. IoC упрощает тестирование и управление зависимостями в приложении.
- **REST API**: Разработка RESTful сервисов с использованием HTTP-методов, структурирование API, обработка запросов и ответов.
- **Многослойная архитектура**: Разделение приложения на слои (контроллеры, сервисы, репозитории, модели), обеспечивающее модульность и тестируемость.
- **HTTP и клиент-серверная архитектура**: Понимание клиент-серверной архитектуры, методов HTTP (GET, POST, PUT, DELETE), влияния TCP/IP handshake и bandwidth-delay product на скорость запросов.

### 2. Безопасность
- **JWT (JSON Web Tokens)**: Реализация аутентификации и авторизации с использованием JWT токенов, включая генерацию, проверку и обновление токенов.
- **Spring Security**: Настройка безопасности приложения, включая фильтры аутентификации, разграничение доступа по ролям, защиту от CSRF-атак.
- **UserDetailsService**: Использование интерфейса UserDetailsService для загрузки данных пользователя по имени пользователя.
- **Хэширование паролей**: Сравнение алгоритмов хэширования (bcrypt vs SHA-256), понимание, почему bcrypt предпочтительнее для хранения паролей.
- **CSRF защита**: Понимание атак типа Cross-Site Request Forgery и реализация защиты с использованием CSRF токенов.
- **Сессии и Cookies**: Работа с сессиями и куками, понимание разницы между stateful (сессии) и stateless (JWT) аутентификацией.
- **Управление сессиями**: Реализация механизма сессий с использованием cookie и токенов.

### 3. Управление данными
- **JPA/Hibernate**: Работа с реляционными базами данных, создание сущностей, репозиториев, выполнение CRUD-операций.
- **CRUD-операции**: Понимание основных операций (Create, Read, Update, Delete), паттерна DAO и разницы между PUT и POST в REST (идемпотентность).
- **ACID-свойства транзакций**: Понимание свойств атомарности, согласованности, изолированности и устойчивости транзакций.
- **PostgreSQL**: Использование PostgreSQL в качестве основной базы данных, настройка подключения, миграции схемы.
- **N+1 проблема**: Понимание проблемы N+1 запросов и методы ее решения.
- **Кэширование**: Реализация кэширования с использованием Spring Cache и Caffeine для повышения производительности.
- **Пагинация**: Использование пагинации для эффективной работы с большими объемами данных.

### 4. Взаимодействие с внешними системами
- **Telegram Bot API**: Интеграция с Telegram для отправки уведомлений пользователям.
- **Файловая система**: Загрузка, хранение и скачивание файлов с сервера, обработка MIME-типов и multipart/form-data.
- **CSV-импорт**: Парсинг и импорт данных из CSV-файлов.

### 5. Генерация отчетов
- **PDF-генерация**: Создание PDF-отчетов с использованием iText7.
- **Отчеты разных типов**: Ежедневные, еженедельные, ежемесячные и пользовательские отчеты.

### 6. Тестирование
- **Юнит-тесты**: Написание тестов для отдельных компонентов приложения.
- **Интеграционные тесты**: Тестирование взаимодействия между компонентами и работу с базой данных.
- **Тестирование безопасности**: Проверка корректности аутентификации и авторизации.
- **JUnit и Mockito**: Использование фреймворков для написания тестов, понимание видов тестов (unit vs integration).

### 7. Контейнеризация и развертывание
- **Docker**: Создание Docker-образов для приложения и зависимостей.
- **Docker Compose**: Оркестрация нескольких сервисов (приложение, база данных, Redis).

### 8. Веб-серверы и многопоточность
- **Модели threading**: Сравнение моделей threading в веб-серверах (threads per connection vs thread pool).
- **Race condition и Amdahl's law**: Понимание проблем многопоточности и законов масштабирования.
- **ExecutorService**: Использование для оптимизации многопоточных операций.

### 9. Spring MVC и параметры запросов
- **@RequestParam и @PathVariable**: Использование для обработки параметров запросов.
- **Безопасность параметров запросов**: Понимание, почему пароли не передаются в GET-параметрах, обработка кодирования URL и предотвращение уязвимостей.

### 10. Обработка файлов в Spring Boot
- **Обработка статических файлов**: Настройка обработки статических ресурсов в Spring Boot.
- **MIME-типы и multipart/form-data**: Понимание формата multipart/form-data и важности проверки MIME-типов для безопасности.
- **Загрузка файлов**: Обработка загрузки файлов и обеспечение безопасности при работе с разными расширениями.

### 11. Инструменты разработки
- **Maven vs Gradle**: Сравнение инструментов сборки проектов, структура типичного Maven-проекта.
- **Форматы данных**: Сравнение XML, JSON и YAML в контексте хранения данных в веб-API.
- **Настройка VS Code**: Настройка среды разработки для работы с Java.

## Применение в будущем
- **Разработка масштабируемых приложений**: Полученные знания о многослойной архитектуре, кэшировании и решении проблемы N+1 помогут при создании высоконагруженных приложений.
- **Безопасность приложений**: Понимание принципов аутентификации и авторизации, хэширования паролей (bcrypt), CSRF защиты и разницы между stateful и stateless аутентификацией будет полезно при разработке защищенных систем.
- **Интеграция с внешними сервисами**: Опыт интеграции с Telegram и другими API пригодится при создании приложений с расширенной функциональностью.
- **Многопоточность и производительность**: Знания о моделях threading, race condition и использовании ExecutorService помогут при оптимизации производительности приложений.
- **Управление проектами**: Знания о структуре приложения и взаимодействии компонентов помогут при планировании и реализации проектов.
- **Разработка API**: Опыт создания REST API с использованием различных HTTP-методов, понимание идемпотентности и безопасности параметров запросов будет полезен при разработке сервисов, предназначенных для интеграции с другими системами.
- **Автоматизация процессов**: Навыки работы с файлами, импортом данных и генерацией отчетов пригодятся при создании систем автоматизации бизнес-процессов.
- **Работа с базами данных**: Понимание ACID-свойств транзакций, паттерна DAO и оптимизации запросов будет полезно при разработке приложений с высокими требованиями к надежности и производительности.
- **Инструменты разработки**: Знания о Maven, форматах данных (XML, JSON, YAML) и настройке среды разработки пригодятся при работе с различными проектами и технологиями.

## Заключение

Курс системного программирования предоставил ценный опыт в области разработки современных веб-приложений. Реализованный проект системы управления автобусами и уведомлениями о событиях стал практической демонстрацией применения изученных концепций и технологий.

В процессе курса были освоены ключевые навыки:
- Разработка REST API с использованием Spring Boot и понимание HTTP-протокола
- Обеспечение безопасности приложений с помощью JWT, Spring Security, хэширования паролей и CSRF защиты
- Работа с базами данных, транзакциями, решение проблемы N+1 и оптимизация запросов
- Кэширование данных и пагинация
- Интеграция с внешними сервисами
- Генерация отчетов и работа с файлами
- Тестирование приложений (юнит и интеграционные тесты)
- Контейнеризация и деплой приложений
- Многопоточность и оптимизация производительности
- Работа с параметрами запросов и безопасность веб-приложений

Лекционные материалы обеспечили глубокое понимание архитектурных принципов и современных подходов к разработке программного обеспечения.

Полученные знания и навыки станут прочной основой для дальнейшего профессионального развития в области разработки программного обеспечения. Они позволят создавать надежные, безопасные и масштабируемые приложения, соответствующие современным требованиям.
